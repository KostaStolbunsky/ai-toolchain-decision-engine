# Docs

This directory contains project-level documentation: vision, open questions, session notes, and security design.

## Contents

| Location | Description |
|---|---|
| [`vision.md`](vision.md) | Long-term product vision and strategic direction |
| [`project.md`](project.md) | Project scope, status, and delivery plan |
| [`open-questions.md`](open-questions.md) | All open decisions and unresolved topics |
| [`security/`](security/) | Security design and threat model |
| [`session-notes/`](session-notes/) | Per-session working notes for continuity |

## open-questions.md

This is the single place for all open questions across the project. Questions found in other files should be moved here. Each question should have an owner, a priority, and a target resolution date where possible.
