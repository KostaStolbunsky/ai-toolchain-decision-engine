# Meta-Model

This directory contains the abstract model that powers the decision engine.

## Philosophy

The model is **technology-agnostic at its core**. It defines abstract roles, criteria and relationships — not specific products or vendors. Tools and platforms are registered separately as implementations of abstract node roles.

This separation ensures the engine remains valid even as the AI toolchain market evolves.

## Contents

- `nodes.md` — Abstract node type definitions
- `criteria.md` — Evaluation criteria schema
- `use-cases.md` — Use case taxonomy (abstract)

> ⚠️ All files in this directory are **drafts**. The model is being iteratively defined through discussion and research.
